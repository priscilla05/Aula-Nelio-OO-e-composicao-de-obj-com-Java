# Aula-Nelio-OO-e-composicao-de-obj-com-Java
##Um projetinho simples para treinar e também relembrar POO e a composição de objetos em Java

###Implementação de  um programa de console conforme enunciado:
###Ler os dados de um pedido com N itens (N fornecido pelo usuário). Depois, mostrar um sumário do pedido conforme exemplo (próxima página). ###Nota: o instante do pedido deve ser o instante do sistema.
Exemplo:
Enter cliente data:
Name: Vitoria
Email: vivi@gmail.com
Birth date (DD/MM/YYYY): 07/03/2000
Enter order data:
Status: PROCESSING
How many items to this order? 2
Enter #1 item data:
Product name: TV
Product price: 1000.00
Quantity: 1
Enter #2 item data:
Product name: Mouse
Product price: 40.00
Quantity: 2

ORDER SUMMARY:
Order moment: 20/04/2018 11:25:09
Order status: PROCESSING
Client: Vitoria (07/03/2000) - vivi@gmail.com
Order items:
TV, $1000.00, Quantity: 1, Subtotal: $1000.00
Mouse, $40.00, Quantity: 2, Subtotal: $80.00
Total price: $1080.00
